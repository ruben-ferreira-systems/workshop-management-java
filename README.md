# Workshop Job Manager

A Java application for managing workshop job sheets and repair orders.

This project was developed as a practical software engineering project to demonstrate object-oriented programming, clean code principles and software architecture.

## Features

- Create job sheets
- Register customers
- Register vehicles
- Manage repair orders
- Track repair status
- Search records
- Generate reports *(if applicable)*

## Technologies

- Java
- PostgreSQL

## Learning Objectives

This repository demonstrates:

- Object-Oriented Programming
- Encapsulation
- Inheritance
- Polymorphism
- Exception Handling
- Layered Architecture
- Data Persistence

## Educational Purpose

This project is shared as a reference implementation for educational purposes.

It represents one possible solution to the problem and is intended to help students understand software design and project organization rather than provide a template for direct reuse.
